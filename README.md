## EndoTrack

Demo: https://endo-track-woad.vercel.app/

# What It Is

EndoTrack is a personalized care tool designed to help people with endometriosis track symptoms and turn their daily experiences into meaningful health insights.

# Why I Made It

Got awarness about Endometriosis through a TikTok creator and it was so heart breaking to hear that research on males getting balder as they age was getting more funding than a chronic condition. So made this to look at the early signs and help educate women on it.

# Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
